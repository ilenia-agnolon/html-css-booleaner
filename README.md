### # Overview

Exercise focused on reproducing a magazine-style article layout using structured HTML and modular CSS, working one block at a time and committing incrementally with Git.

### # Assignment Summary

The exercise required to:

- Recreate the provided layout using the same approach shown in class
- Build the page step by step, adding CSS rules progressively
- Create the project folder `html-css-booleaner`
- Open the folder with VS Code and create `index.html`
- Use lowercase file names and hyphens instead of spaces
- Create an `img` folder for all required images
- Create a `css` folder and add the stylesheet
- Publish the repository on GitHub using VS Code
- Work incrementally and push changes regularly

Bonus:

- Starting from the same HTML, create a second CSS file to produce an alternative visual version
- The alternative version should have a narrower, centered page body while keeping the header background full-width
